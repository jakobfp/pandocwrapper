# pandocwrapper

Minimal wrapper for pandoc. Used only for own purposes.

### Requirements

Full list in [requirements](./requirements.txt)

### Start API

`$ python app.py` is enough to let the application serve at [http://localhost:5000/api](http://localhost:5000/api).
